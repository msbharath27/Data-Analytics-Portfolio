# Bike Sales Data Analysis (Excel)

## Objective
To perform basic data entry, formatting, and calculations in Excel to analyze revenue and profit from product sales.

## Tools Used
- Microsoft Excel

## Key Steps Performed
- Entered structured tabular data
- Formatted column headers and currency columns
- Used formulas for:
  - Total Cost = Order_Quantity × Unit_Cost
  - Revenue = Order_Quantity × Unit_Price
  - Profit = Revenue − Total_Cost
- Used SUM functions to calculate total revenue and profit

## Skills Demonstrated
- Data entry and formatting
- Basic Excel formulas
- SUM function usage
- Practical understanding of cost → revenue → profit logic
